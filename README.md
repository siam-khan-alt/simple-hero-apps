# Simple Hero Apps

## Description
Simple Hero Apps is a responsive React application that showcases various mobile apps.  
Users can view app details, install/uninstall apps, and check ratings and download statistics.

---

## Technologies Used
- React
- React Router v7
- TailwindCSS v4
- DaisyUI
- Recharts
- Axios
- React-Toastify
- React Spinners
- Vite
- LocalStorage
